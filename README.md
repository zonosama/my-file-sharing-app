# 📁 マイファイル共有サイト（完全版）

Streamlitで作成した多機能ファイル共有サイトです。

## ✨ 主要機能

### 🔐 パスワード認証
- ユーザー登録・ログイン機能
- パスワードはSHA-256でハッシュ化
- 複数ユーザーのサポート

### 📤 ファイルアップロード
- 複数ファイルを同時アップロード
- プログレスバー表示
- ファイル名の重複自動回避

### 📝 ファイルのメモ・タグ機能
- 各ファイルにメモを記録
- タグ付けで分類管理
- タグでフィルタリング可能

### 🗂️ フォルダ分類機能
- ファイルをフォルダで整理
- フォルダ別表示
- フォルダ名の変更可能

### 🔗 ファイル共有リンク生成
- ワンクリックで共有リンクを生成
- 認証なしでダウンロード可能
- 他の端末・他の人とファイル共有

### 🔍 高度な検索・フィルタ機能
- ファイル名で検索
- フォルダでフィルタリング
- タグでフィルタリング
- 複数条件の組み合わせ可能

### 📊 統計情報
- 総ファイル数
- 総サイズ
- フォルダ数
- 共有リンク一覧

## セットアップ

### 1. 仮想環境のアクティベート

```bash
# Windows
.venv\Scripts\activate

# macOS/Linux
source .venv/bin/activate
```

### 2. パッケージのインストール

```bash
pip install -r requirements.txt
```

### 3. アプリケーションの起動

```bash
streamlit run app.py
```

ブラウザが自動的に開き、`http://localhost:8501`でアクセスできます。

## 使い方

### 初回起動時

1. 「新規登録」をクリック
2. ユーザー名とパスワードを入力
3. 登録完了後、ログイン

### ファイルのアップロード

1. **アップロードタブ**を開く
2. 保存先フォルダを選択（または新規作成）
3. ファイルを選択
4. メモやタグを入力（任意）
5. 「アップロード実行」をクリック

### ファイルの管理

1. **ファイル一覧タブ**を開く
2. 検索、フォルダ、タグでフィルタリング
3. ダウンロード、編集、削除が可能

### ファイルの共有

1. **ファイル一覧タブ**で共有したいファイルを見つける
2. 「🔗 共有リンク」ボタンをクリック
3. 生成されたURLをコピーして相手に送信
4. 相手はログインなしでダウンロード可能

### フォルダの管理

1. **フォルダ管理タブ**を開く
2. フォルダごとのファイル一覧を確認
3. フォルダ名を変更可能

## ⚠️ 注意事項

### Streamlit Cloud使用時の制限

- **一時的なストレージ**: アップロードされたファイルは、アプリの再起動時に消える可能性があります
- **メモリ制限**: 1GB（大量のファイルアップロードには注意）
- **パブリックアクセス**: デプロイしたアプリは誰でもアクセス可能

### セキュリティについて

- パスワードはハッシュ化されて保存されます
- 共有リンクは誰でもアクセス可能なので、機密ファイルの共有には注意
- Streamlit Cloudでは、リポジトリがPublicの場合、コードも公開されます

### 別端末間でのファイル共有

このアプリをStreamlit Cloudにデプロイすれば：
- ✅ 任意の端末からアクセス可能
- ✅ 共有リンクで他人とファイル共有可能
- ✅ インターネット経由でどこからでも利用可能

ただし、**一時的な共有**に適しています（永続的な保存には向きません）。

## データ保存について

以下のファイルに情報が保存されます：

- `uploaded_files/`: アップロードされたファイル
- `file_metadata.json`: ファイルのメタデータ（メモ、タグ、フォルダ等）
- `users.json`: ユーザー情報
- `shared_links.json`: 共有リンク情報

## トラブルシューティング

### ログインできない

- ユーザー名とパスワードを確認
- 新規登録からやり直す

### ファイルが消えた

- Streamlit Cloudでは再起動時にファイルが消える可能性があります
- ローカルで実行する場合は永続的に保存されます

### 共有リンクが動作しない

- リンクが正しくコピーされているか確認
- ファイルが削除されていないか確認
